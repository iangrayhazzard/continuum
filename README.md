A human-centric system for time, date, and temperature designed for a unified, global future.

The Hazzard Continuum is an open-source project to rethink how we measure our daily lives. It replaces confusing, inconsistent systems with a set of standards that are logical, predictable, and designed for a future where global collaboration is the norm.

## The Core Components

The Continuum is built on three pillars, each designed to simplify a part of your day.

### 1. The Continuum Calendar

A rational and perennial calendar system.

-   **13 Months, 28 Days:** Every month has exactly four weeks, starting on a Sunday and ending on a Saturday.
-   **Perennial:** Your birthday is on the same day of the week, every single year. The 15th of any month is always a Sunday. Planning becomes trivial.
-   **New Year on March 1st:** The calendar year aligns with the beginning of Spring.
-   **Gormanuary & Intermission:** A 13th month, "Gormanuary," completes the \(13 \times 28 = 364\) days. A special year-end holiday called "Intermission" (or two on leap years) sits outside the weeks and months, bringing the total to 365.

### 2. Universal Time

One time for the entire world.

-   **No More Timezones:** The entire world uses a single 24-hour clock based on Coordinated Universal Time (UTC). When it's 14:00 in London, it's 14:00 in New York and Tokyo.
-   **Future-Proof:** As the world "shrinks" with high-speed travel and constant online collaboration, a single time standard eliminates confusion.
-   **Human-Centric Context:** The app helps you understand Universal Time in your local context by showing you your local sunrise, solar noon, and sunset times in UTC.

### 3. Hazzen Temperature

A temperature scale based on what it actually feels like.

-   **Centered on Comfort:** \(0 \text{ H}\) is a comfortable room temperature (\(66^\circ\text{F}\)).
-   **Intuitive Scale:** Positive values are warm (\(+5 \text{ H}\) is a hot day). Negative values are cold (\(-5 \text{ H}\) is chilly). The scale is based on what you'd wear.
-   **Simple Conversion:** Each degree of Hazzen (\(1 \text{ H}\)) is equal to \(4^\circ\text{F}\).

## Getting Started (For Developers)

Want to run the web app locally?

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/continuum.git
    cd continuum
    ```
2.  **Navigate to the web app:**
    ```bash
    cd apps/webapp
    ```
3.  **Install dependencies and run:**
    ```bash
    npm install
    npm run dev
    ```
